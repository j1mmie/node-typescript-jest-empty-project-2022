# node-typescript-jest-empty-project-2022 

Directory structure:
 - dist
 - src
 - tests

Includes:
 - Node 18.6
 - Typescript 4.8.4
 - Jest 29.2.2

Base config: [node18/tsconfig.json](https://github.com/tsconfig/bases/blob/main/bases/node18.json)

Run tests:
```
npx jest
```

Compile:
```
npx tsc
```

Execute:
```
node dist/index.js
```
